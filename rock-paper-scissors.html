About the Project

Project Description

Rock, Paper, Scissors is a turn-based decision game transformed into a fully interactive digital application. The player competes against a computer opponent over multiple rounds, with results tracked in real time. Built using core OOP concepts in C++, this project bridges theory with practical software development.

🎮

What is it?

A digital version of the classic Rock, Paper, Scissors game where a human player competes against a computer in multiple rounds.

💻

How is it built?

Built using C++ with Object-Oriented Programming principles including classes, objects, constructors, and methods.

🤖

The AI Opponent

The computer randomly generates its choice each round using a random number generator, ensuring fair and unbiased gameplay.

📊

Score System

Scores are tracked across all rounds and a final winner is declared at the end of the game based on total round wins.

What's Included

Key Features

Every feature has been designed to make the game functional, user-friendly, and academically relevant.

Gameplay 🎯

Interactive Gameplay

Player selects Rock, Paper, or Scissors each round through a simple console input system.

AI 🤖

Computer Opponent

The computer randomly picks a choice every round, ensuring unpredictable and fair competition.

Scoring 🏆

Score Tracking

Wins and losses are recorded and displayed after every round so the player always knows the current standing.

Rounds 🔄

Custom Rounds

Player can choose how many rounds to play before the game begins, making it flexible and replayable.

Result 📋

Instant Feedback

After each round, the result (Win / Lose / Draw) is immediately displayed along with both choices made.

Final 🎖️

Final Result

At the end of all rounds, the overall winner is announced based on the total score accumulated.

Goals

Project Objectives

This project was designed with clear academic and technical goals in mind.

01

Build a Functional Game

Design and develop a complete Rock, Paper, Scissors game that runs correctly from start to finish without errors.

02

Apply OOP Concepts

Use Object-Oriented Programming principles such as classes, objects, constructors, and methods in a real project.

03

Implement Game Logic

Write accurate win, lose, and draw conditions based on the standard rules of Rock, Paper, Scissors.

04

Handle User Input

Accept and validate player input properly, handling invalid entries without crashing the program.

05

Display Clear Output

Show round results, scores, and the final winner in a clean, readable format after each round and at game end.

06

Demonstrate Programming Skills

Produce a well-structured, working C++ program that reflects understanding of programming fundamentals learned in class.
